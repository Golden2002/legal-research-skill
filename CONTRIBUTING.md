# 贡献指南

感谢您对本项目的兴趣！我们欢迎任何形式的贡献，包括但不限于：

- 🐛 报告 Bug
- 💡 提出新功能建议
- 📝 完善法律检索逻辑
- 📚 补充更多法律领域速查表
- 🔧 改进文档

## 如何贡献

### 1. Fork 本仓库

点击仓库页面右上角的 "Fork" 按钮。

### 2. 克隆到本地

```bash
git clone https://github.com/your-username/legal-research-skill.git
cd legal-research-skill
```

### 3. 创建分支

```bash
git checkout -b feature/your-feature-name
```

### 4. 进行修改

请确保您的修改：
- 符合现有的代码风格
- 不破坏现有功能
- 添加适当的测试（如果有）

### 5. 提交修改

```bash
git add .
git commit -m "Add: your feature description"
```

### 6. 推送分支

```bash
git push origin feature/your-feature-name
```

### 7. 创建 Pull Request

在 GitHub 上创建 Pull Request，描述您的修改内容。

## Pull Request 指南

### 标题格式

- `Add:` 新功能
- `Fix:` Bug 修复
- `Update:` 更新现有功能
- `Docs:` 文档改进
- `Refactor:` 代码重构

### 内容要求

请在 PR 描述中包含：

1. **修改目的**：为什么要做这个修改？
2. **修改内容**：具体改了哪些部分？
3. **测试结果**：修改后测试通过了吗？

## 问题反馈

如果您发现了 Bug 或有建议，请先搜索是否已有相关 issue。如果没有，请创建一个新的 issue 并包含：

- 清晰的标题
- 详细的描述
- 复现步骤（如果是 Bug）
- 您的环境信息

## 代码规范

### SKILL.md 格式要求

- 使用标准的 Markdown 格式
- 法条引用必须准确
- 保持 YAML front matter 格式正确
- 触发条件描述要清晰

### 法条引用规范

- 法律：《中华人民共和国XX法》第X条
- 司法解释：法释〔XXXX〕XX号第X条
- 行政法规：《XX条例》第X条
- 地方性法规：《XX省XX条例》第X条

## 联系方式

如果您有任何问题，欢迎通过以下方式联系我们：

- GitHub Issues
- GitHub Discussions
