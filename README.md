# Legal Research SKILL

中国法律全领域系统性检索专家 - AI Agent Skill for Legal Research

## 📖 项目简介

本项目提供一套专业的法律检索 Agent Skill，可用于 Cursor、Claude Code、OpenCode 等 AI 编程工具。无论是非法律专业人士需要了解法律知识，还是法律从业者进行法律检索，都能提供系统性的法律规范查找与整理服务。
> P.S. : 直接复制`SKILL.md`中的内容，作为提示词输入给对话式ai，比如豆包、deepseek也是好用的

## 🎯 核心功能

- 🔍 **全领域法律检索**：覆盖劳动争议、合同纠纷、侵权责任、婚姻家庭、行政法、刑事等领域
- 📚 **法源全面覆盖**：法律、行政法规、司法解释、部门规章、地方性法规、指导性案例
- 📋 **结构化输出**：提供标准化的法律检索报告模板
- ⏰ **时效性处理**：诉讼时效、除斥期间等期限梳理
- 📖 **引用规范**：标准法条引用格式

## 快速开始

### 前置依赖

- Node.js 20.6+（含 npm）
- Git

### 安装 OpenSkills

```bash
# 使用 npx 直接运行
npx openskills --version

# 或全局安装
npm i -g openskills
openskills --version
```

### 安装本 Skill

```bash
# 方式一：通过 OpenSkills 安装
npx openskills install your-username/legal-research-skill

# 方式二：手动克隆
git clone https://github.com/your-username/legal-research-skill.git
# 将 skill 复制到对应工具的 skills 目录
```

## 核心指令

### 1. 严格依据成文规范
只引用法律、行政法规、司法解释、部门规章、地方性法规等公开文本，不参考学术观点、未生效草案或非官方解读。

### 2. 全面覆盖法源
- **法律**：全国人大及其常委会制定
- **行政法规**：国务院制定
- **司法解释**：最高人民法院、最高人民检察院发布
- **部门规章**：国务院各部委制定
- **地方性法规/地方政府规章**：省级及设区市人大、政府制定
- **指导性案例**：最高人民法院发布的指导性案例（具有参照效力）
- **典型案例**：最高人民法院发布的典型案例（仅供参考）

### 3. 专业处理逻辑
- 识别"一般规定 vs 特别规定"，说明适用优先级
- 标注法条间引用关系
- 联动法律与配套规范
- 若涉及地域差异，标注地方性规定的适用范围
- 区分新旧法效力，明确法律变更时间节点

## 工作流程

### 第一步：问题解析
在开始检索前，如用户未提供以下信息，请主动要求：
- **（1）争议类型**：如"劳动争议""商品房买卖合同纠纷"
- **（2）关键事实要素**：包括主体类型、行为性质、管辖地域、时间节点
- **（3）检索目标**：如"责任认定依据""赔偿标准计算"

### 第二步：系统性检索
按效力层级从高到低列举相关规范：
1. 法律层面
2. 行政法规/司法解释
3. 部门规章
4. 地方性规定

### 第三步：规范冲突与适用分析
说明《立法法》规定的适用规则，结合司法解释说明司法实践倾向

### 第四步：时效与期限梳理
如涉及时效问题，需单独梳理各类期限

### 第五步：输出结构化文档
按标准模板输出法律检索报告

## 常见法律领域速查

### 劳动争议

| 事项 | 法律规定 | 要点 |
| :--- | :--- | :--- |
| 违法解除赔偿金 | 《劳动合同法》第87条 | 2N = 经济补偿标准×2 |
| 经济补偿N | 《劳动合同法》第46、47条 | 每满1年=1个月工资 |
| 代通知金 | 《劳动合同法》第40条 | 1个月工资 |

### 合同纠纷

| 事项 | 法律规定 | 要点 |
| :--- | :--- | :--- |
| 定金罚则 | 《民法典》第586、587条 | 双倍返还/无权请求返还 |
| 违约金 | 《民法典》第585条 | 约定违约金 |

### 侵权责任

| 事项 | 法律规定 | 要点 |
| :--- | :--- | :--- |
| 人身损害赔偿 | 《民法典》第1179条 | 医疗费、护理费、误工费等 |
| 精神损害赔偿 | 《民法典》第1183条 | 严重精神损害可主张 |

## 触发条件

### 高优先级（必须触发）

- 明确要求查找法条/法规/司法解释/规范性文件
- 要求判断合法性/违法性（"违法吗""合法吗"）
- 要求查找赔偿标准/补偿金额/责任认定/程序要求
- 询问"根据什么法律""法律怎么规定"

### 中优先级（结合上下文触发）

- "怎么办""怎么维权""能不能告"
- "需要什么手续""要具备什么条件"
- "还能主张什么""可以去哪里投诉"

### 不触发情况

- 仅询问法律概念/名词解释
- 仅要求推荐律师/法律服务
- 仅讨论法律新闻/案例故事
- 仅询问法律考试/学习问题
